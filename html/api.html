jDan734's API help

Parameters

action:
Which action to perform
ban:
Send info about ban: time and status
random:
Send random number from 0 to 1000
limit:
Change limit
  • default: 10
  • max: 10000
format:
The format of the output.
json:
Output data in JSON format (default)

Examples

  • Send random number (limit 126)
    api? action=random & limit=126
  • Send information about ban
    api? action=ban
